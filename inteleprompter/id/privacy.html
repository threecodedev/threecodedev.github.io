inTelePrompter — Kebijakan Privasi

Berlaku Juli 2026 · ThreeCode

Versi singkatnya: inTelePrompter tidak mengumpulkan data. Sama sekali tidak. Tidak ada akun, tidak ada analitik, tidak ada pelacakan, tidak ada iklan, dan tidak ada SDK pihak ketiga. Naskah Anda, pengaturan Anda, dan suara Anda tidak pernah meninggalkan perangkat Anda.

Apa yang kami kumpulkan

Tidak ada. inTelePrompter tidak memiliki server, tidak memerlukan akun, dan tidak mengirim informasi apa pun — pribadi atau lainnya — kepada kami atau kepada siapa pun. Aplikasi bekerja sepenuhnya offline.

Naskah dan pengaturan Anda

Semua yang Anda buat di inTelePrompter — naskah, preferensi tampilan, tema, hasil kalibrasi — hanya disimpan di perangkat Anda, menggunakan penyimpanan aplikasi standar Apple. Jika Anda menghapus aplikasi, data tersebut ikut terhapus. Kami tidak pernah melihatnya.

Mikrofon

Dua fitur opsional menggunakan mikrofon, dan keduanya memproses audio sepenuhnya di perangkat Anda:

  • Jeda Cerdas menggunakan deteksi aktivitas audio untuk mengetahui saat Anda berhenti membaca, sehingga naskah dapat menggulung balik ke titik di mana Anda terakhir berbicara. Audio Anda tidak pernah direkam, tidak pernah disimpan, tidak pernah ditransmisikan, dan tidak pernah dibandingkan dengan konten naskah Anda.
  • Perintah suara Halo Telly menggunakan framework ucapan di perangkat milik Apple untuk mengenali sekumpulan kecil frasa perintah tetap (seperti «putar», «jeda», dan «mundur»). Hanya frasa perintah pendek ini yang dikenali — tidak pernah konten naskah Anda. Pengenalan berjalan di perangkat Anda; suara Anda tidak pernah dikirim kepada kami, ke server Apple, atau ke mana pun.

Kedua fitur bersifat opsional, dapat dinonaktifkan di Pengaturan kapan saja, dan aplikasi bekerja sepenuhnya tanpa keduanya.

Kamera dan Foto

Jika Anda memilih untuk merekam diri sambil membaca, video disimpan langsung ke pustaka Foto Anda dan tidak ke tempat lain. inTelePrompter memiliki akses Foto tambah-saja: aplikasi dapat menyimpan video baru, tetapi tidak pernah dapat membaca, menjelajahi, atau memodifikasi foto atau video Anda yang sudah ada.

File yang Anda impor

Saat Anda mengimpor naskah melalui pemilih Files, teks file disalin ke pustaka naskah di perangkat Anda. Impor hanya terjadi saat Anda memulainya, dan kontennya tetap di perangkat Anda.

Pembelian

inTelePrompter adalah pembelian satu kali tanpa pembelian dalam aplikasi dan tanpa langganan. Pembelian Anda diproses sepenuhnya oleh Apple; kami tidak menerima informasi pembayaran apa pun tentang Anda.

Anak-anak

Karena inTelePrompter tidak mengumpulkan data dari siapa pun, aplikasi ini tidak mengumpulkan data dari anak-anak.

Perubahan pada kebijakan ini

Jika versi aplikasi di masa mendatang mengubah cara data ditangani, kami akan memperbarui kebijakan ini sebelum versi tersebut dirilis dan mencatat perubahannya di sini. Janji di atas — tanpa pengumpulan, tanpa pelacakan, hanya di perangkat — adalah fondasi bagaimana inTelePrompter dibangun.
